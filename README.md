# Calculadora Python

## Descripción
Este es mi **primer proyecto en Python**. Se trata de una calculadora sencilla que permite realizar operaciones básicas: sumar, restar, multiplicar y dividir.  

El programa se ejecuta en consola y permite al usuario realizar varias operaciones hasta que decida salir.

---

## Funcionalidades
- Suma, resta, multiplicación y división.  
- Evita la división entre cero.  
- Permite repetir operaciones múltiples veces hasta que el usuario decida salir.  

---

## Tecnologías
- Python 3  
- Consola/terminal  
- Bucle `while`, condicionales `if/elif/else`
